GitHub - Projet 4

Classification automatique des biens de consommation

Description : Développement d'un moteur de classification d'articles pour une marketplace e-commerce, automatisant l'attribution de catégories basées sur les images et descriptions des produits. GitHub - Projet 5

Mise en oeuvre d'un outil de prédiction de remboursement de crédit

Description : Développement d'un outil de scoring crédit pour prédire la probabilité de remboursement des clients dans une société financière. Le projet inclut la construction d'un modèle de classification prenant en compte des données variées, un dashboard interactif pour la transparence des décisions d'octroi de crédit, et la mise en production via une API. Une démarche MLOps sera mise en œuvre pour automatiser le cycle de vie du modèle, avec une attention particulière sur le déséquilibre des classes et l'optimisation des coûts d'erreur.

Objectif : Prédire la capacité d'un client à rembourser son prêt, déployer le modèle sous forme d'API et générer un dashboard dans le cloud.

Skills : MLOPS : MLFLOW Tracking, GitHub, Déploiement de modèle (Heroku), Dashboard (DASH), Création API (Flash).
